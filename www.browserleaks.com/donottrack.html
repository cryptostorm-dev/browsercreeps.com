Do Not Track

This tool detect of support in your web-browser the brand new super-duper security privacy tracking protection feature — Do-Not-Track.

The patient's Web-Browser

User-AgentWget/1.15 (mingw32)

HTTP Header method to detect a Do Not Track preference

DNT× Disabled

JavaScript methods to detect a Do Not Track preference

navigator.doNotTrack Disabled
navigator.msDoNotTrack Disabled

This information is also available in other pages of our site. DNT HTTP header you can check at the HTTP Headers Viewer. And the JavaScript based DoNotTrack can be checked looking at the DOM Navigator Object dump.

So what's the point to create a separate page? The only reason why I highlighted this feature in a separate page — it's once again to remind you :

Do-Not-Track does not protect you from privacy invasions.

The system is only recommendatory. By setting the DoNotTrack flag in web-browser, every site will informed about the reluctance of the user to hand over on storage information, in a situation of its usage for tracking users movements online.

Thus, the system only points of disagreement by the display of personalized advertising and makes sense only for sites that voluntarily support this mechanism.
